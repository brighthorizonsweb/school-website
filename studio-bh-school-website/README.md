# Bright Horizons Sanity Studio

Content studio for the Bright Horizons school website.

- **Project ID:** `w4n109hm`
- **Dataset:** `production`

## Setup

```powershell
cd studio-bh-school-website
npm install
npx sanity login
npm run dev
```

Open the Studio at http://localhost:3333.

Create and publish a **Home Page** document with your alert banner, hero title, and welcome text.
